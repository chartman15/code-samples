Open Reading Frame (ORF) Finder & Homology Analysis

This Python script provides tools for identifying Open Reading Frames (ORFs) in DNA sequences from FASTA files and performing simple pairwise homology scoring between ORFs.
It handles sequences on both forward and reverse complement strands and allows filtering ORFs based on a minimum length.
